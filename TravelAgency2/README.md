# Localify Travel Agency Website

Static HTML, CSS and JavaScript website for Localify.

## Structure

- `index.html` - Home page
- `about.html` - Company story, mission, vision, team, achievements and timeline
- `destinations.html` / `destination-detail.html` - Destination listing and guide
- `packages.html` / `package-detail.html` - Tour packages and detail page
- `services.html` - Travel services
- `gallery.html` - Filterable masonry gallery with lightbox
- `blog.html` / `blog-detail.html` - Blog listing and SEO article
- `testimonials.html` - Customer reviews and video placeholders
- `faq.html` - Searchable FAQ accordion
- `contact.html` - Contact form, office details and map
- `assets/css/styles.css` - Full responsive styling
- `assets/js/main.js` - Navigation, filters, forms, counters, dark mode, wishlist and UI behavior
- `robots.txt` and `sitemap.xml` - SEO crawler files

## Run locally

Open `index.html` in a browser, or serve the folder with any static server.

## Deploy to Vercel

1. Push this folder to a Git repository.
2. Import the repository in Vercel.
3. Choose "Other" as the framework preset.
4. Leave build command empty.
5. Set output directory to `.`.
6. Deploy.
